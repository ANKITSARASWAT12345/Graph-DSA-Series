#  Graph Algorithms & DSA Interview Preparation

A complete collection of **Graph Data Structure & Algorithms problems** frequently asked in **FAANG and Top Product-Based Companies**.

This repository contains **carefully curated Graph problems**, detailed explanations, optimized solutions, and interview patterns to help developers master **Graph theory for coding interviews**.

---

#  Topics Covered

This repository covers almost all important **Graph Algorithm patterns** asked in technical interviews.

### Core Graph Concepts

* Graph Representation (Adjacency List / Matrix)
* Directed vs Undirected Graph
* Weighted vs Unweighted Graph
* Sparse vs Dense Graph
* Graph Traversal Techniques

### Graph Traversal

* Breadth First Search (BFS)
* Depth First Search (DFS)

### Shortest Path Algorithms

* Dijkstra Algorithm
* Bellman Ford Algorithm
* Floyd Warshall Algorithm

### Topological Sorting

* Kahn’s Algorithm (BFS)
* DFS Based Topological Sort

### Graph Cycle Detection

* Cycle Detection in Directed Graph
* Cycle Detection in Undirected Graph

### Disjoint Set / Union Find

* Path Compression
* Union by Rank

### Minimum Spanning Tree

* Kruskal Algorithm
* Prim's Algorithm

### Advanced Graph Problems

* Strongly Connected Components
* Kosaraju Algorithm
* Tarjan Algorithm
* Bridges in Graph
* Articulation Points
